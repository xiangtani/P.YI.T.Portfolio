Project

幼兒教育集團系統網站

Description

選用活潑繽紛的漸層色調，做網站整體的色調氛圍，利用虛實整合的手法，將活潑不受拘束、快樂學習的幼兒教育理念，巧妙的帶入網頁中。

因應兩者不同教育方針，利用各式元素整合，鮮明的區分兩者的不同，運用相似的設計手法，避免因為兩者的風格過於迥異，也避免誤導使用者以為自己在不同兩個機構的網站。

    Work Flow

  • 帶領專案進入網站架構流程，獨立執行網站企劃、架構、視覺提案、Mockup與Prototype。
  • 使用Axure繪製Flow與Wirfame進行網站方向確認與溝通。
  • 協助組織官網資訊架構、文案統整及方向。
  • 製作Web Guidelibe、RWD畫面，協助前端工程師參考。
  • 撰寫Html/Sass/RWD響應式靜態網頁呈現。
  • 與PM、前、後端工程師共同執行專案。
