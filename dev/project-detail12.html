Project

新亦勝南北貨品牌官網

Description

迪化街老店品牌翻新，全站利用大量的年貨使用破圖的設計手法，營造出過年氛圍，在下方加入海浪猶如新亦勝氣勢滂礡的改變的決心。

    Work Flow

  • 帶領專案進入網站架構流程，獨立執行網站企劃、架構、視覺提案、Mockup與Prototype。
  • 網站建置前期釐清客戶需求溝通、上線後續後台教育訓練。
  • 使用Axure繪製Flow與Wirfame進行網站方向確認與溝通。
  • 協助組織官網資訊架構、文案統整及方向。
  • 製作Web Guidelibe、RWD畫面，協助前端工程師參考。
  • 撰寫Html/Sass/RWD響應式靜態網頁呈現。
  • 與行銷企劃經理協作，社群平台行銷活動推廣。
  • 攝影企劃、風格蒐集。
  • 專案前中後期客戶溝通協調。
  • 與PM、前、後端工程師共同執行專案。
